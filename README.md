# Task Creation Form

A small practice project: a task creation form built from scratch while preparing for a
junior web developer technical interview.

## What this is

A form for creating a task, with a title field and priority selector, built using
semantic HTML and responsive CSS. This is step one of a larger build — JavaScript
validation and PHP backend integration are planned as follow-up commits.

## What I practiced

- Semantic HTML (`<main>`, `<fieldset>`, `<legend>`, proper `<label>`/`<input>` pairing)
- Flexbox layout and centering
- `:focus` and `:hover` states
- Responsive design with a mobile breakpoint (`@media (max-width: 480px)`)

## How to view it

Clone the repo and open `index.html` directly in a browser — no build step or
dependencies needed.

```bash
git clone git@github.com:tomskezberis/Submission-form.git
cd Submission-form
open index.html   # or just double-click the file
```
